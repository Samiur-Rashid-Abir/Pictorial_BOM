Entry Restricted Area

Cash premises are Entry Restricted area.
This means that only authorized personnel are allowed to enter these areas.
Unauthorized access is strictly prohibited.
Violation may result in disciplinary action.
All access must be logged in the visitor register.
CCTV monitoring is active in all restricted areas.

Access Control

All authorized persons must deactivate the Access Control door by using Security codes/ID card
"Entry Restricted" mark should be displayed on the Access Control door
Access control systems help ensure that only authorized personnel can enter sensitive areas
Maintains security and accountability for all entries and exits
Provides audit trail of all access attempts
Back to Overview

Security Protocols

Whenever any Concern Officer leaves the counter, he/she must:
Lock his/her own cabinet
Lock his/her computer screen by pressing (Windows Key + L)
Inform his/her superior before leaving the counter/premises
These protocols ensure that sensitive information and assets remain secure even when personnel are temporarily away from their stations
Promotes a culture of security awareness and responsibility
Back to Overview

Visitor Register

Any person assigned by BM (other than Key Custodian) wanting to enter the "Cash Vault" must sign in the register
Required information to be recorded:
Purpose of visit
PIN number
Time in
Time out
Department name
All entries must be made in the presence of key custodian(s)
The visitor register provides a clear audit trail of who accessed the cash vault and when, enhancing security and accountability
Back to Overview

Register Maintenance

BM/BOM will maintain this register with his/her opinion
The register should be ready at all times for any inspection
Proper maintenance of registers ensures that all access records are accurately maintained
Readily available for audit purposes
Ensures compliance with regulatory requirements
Helps in tracking and analyzing access patterns over time
Back to Overview

Access Deactivation

All authorized persons must deactivate the Access Control door by using Security codes/ID card
Proper deactivation ensures that the access control system functions as intended
Maintains the security integrity of the cash premises
Creates a log of all deactivation events for security monitoring
Ensures only current authorized personnel can access restricted areas
Back to Overview

CCTV Coverage

Ensure CCTV coverage in every area/corner of cash area including:
HT (Head Teller) areas
Cash sorter desk
DVR time should be matched with real time
Comprehensive CCTV coverage provides visual monitoring and recording of all activities in cash areas
Serves as both a deterrent and an investigative tool
Ensure proper storage and backup of CCTV footage as per regulatory requirements
Regular maintenance and testing of CCTV systems is mandatory
