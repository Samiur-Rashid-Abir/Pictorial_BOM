Entry Restricted Area

Cash premises are Entry Restricted area.

This means that only authorized personnel are allowed to enter these areas. Unauthorized access is strictly prohibited and may result in disciplinary action.

Access Control

All authorized person must deactivate the Access Control door by using Security codes/ID card.

"Entry Restricted" mark should be hanged on the Access Control door.

Access control systems help ensure that only authorized personnel can enter sensitive areas, maintaining security and accountability.

Back to Overview

Security Protocols

Whenever any Concern Officer leaves the counter, he/she must:

  • Lock his/her own cabinet
  • Lock his/her computer screen by pressing (Windows Key+ L)
  • Inform his/her superior before leaving the counter/premises

These protocols ensure that sensitive information and assets remain secure even when personnel are temporarily away from their stations.

Back to Overview

Visitor Register

Any person assigned by BM (other than Key Custodian) wants to enter in "Cash Vault" must sign in the register with:

  • Purpose
  • PIN
  • Time in
  • Time out
  • Dept. name

All entries must be made in the presence of key custodian(s).

The visitor register provides a clear audit trail of who accessed the cash vault and when, enhancing security and accountability.

Back to Overview

Register Maintenance

BM/BOM will maintain this register with his/her opinion.

The register should be ready at all the time for any inspection.

Proper maintenance of registers ensures that all access records are accurately maintained and readily available for audit purposes.

Back to Overview

Access Deactivation

All authorized person must deactivate the Access Control door by using Security codes/ID card.

Proper deactivation ensures that the access control system functions as intended, maintaining the security of the cash premises.

Back to Overview

CCTV Coverage

Ensure CCTV coverage in every area/corner of cash area including:

  • HT (High Tension) areas
  • Cash sorter desk

DVR time should be matched with real time.

Comprehensive CCTV coverage provides visual monitoring and recording of all activities in cash areas, serving as both a deterrent and an investigative tool.
